eDM Helper Bookmarklets

Kumpulan bookmarklet berguna untuk daily work Anda

Cara Menggunakan Bookmarklet:

  1. Drag tombol bookmarklet di bawah ini ke bookmark bar browser Anda
  2. Alternatif: Klik kanan → Add to Bookmarks
  3. Buka halaman target dan klik bookmarklet untuk menjalankan

Catatan: Works on Chrome, Firefox, Edge, Safari. Pastikan bookmark bar visible (Ctrl+Shift+B).

Shortcut Keyboard Berguna:

Ctrl + U View source code
Ctrl + Shift + C Inspect element
Ctrl + Shift + J Open console
Ctrl + Shift + B Toggle bookmark bar
Ctrl + A Select all
Ctrl + C Copy

Pro Tips:

  • Bookmarklets bekerja lebih baik di modern browsers (Chrome, Firefox, Edge, Safari 12+)
  • Beberapa website mungkin memblokir fetch API - gunakan view source sebagai alternatif
  • Simpan bookmarklets di folder khusus untuk organisasi yang lebih baik
  • Test bookmarklets di development environment terlebih dahulu
